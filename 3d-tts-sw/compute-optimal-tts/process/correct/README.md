# 题目正确性识别工具

识别模型输出中哪些题目做对了，哪些做错了。

## 使用方法

```bash
python identify_correct_questions.py <输出目录> [-o 输出文件] [--verbose]
```

### 示例

```bash
# 基本用法
python identify_correct_questions.py ../../src/output/.../16384_4_1

# 保存到文件
python identify_correct_questions.py ../../src/output/.../16384_4_1 -o result.json
```

## 输出示例

```
======================================================================
题目正确性统计
======================================================================

✅ 做对的题目 (majority_vote=1):
   题号: [1, 3, 6, 15, 19]
   总计: 5/30
   正确率: 16.67%

❌ 做错的题目 (majority_vote=0):
   题号: [0, 2, 4, 5, 7, ...]
   总计: 25/30

======================================================================
```

## 参数说明

- `output_dir`: 输出目录路径
- `-o, --output`: 结果输出文件（可选）
- `--verbose`: 显示详细信息（可选）
